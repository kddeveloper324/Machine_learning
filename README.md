## üèè IPL 2025: Ultimate Victory Predictor

This project is a Machine Learning web application designed to predict the winner of IPL matches with a calculated confidence score. It leverages historical data and team performance metrics to analyze matchups in real-time.

### üöÄ Key Features

* **Match Simulation:** Select Home vs. Away teams to simulate a match scenario.
* **Condition Analysis:** Inputs for specific match venues and toss outcomes (winner & decision) to refine predictions.
* **Predictive Modeling:** Returns a predicted winner along with a **Model Confidence Score** (e.g., 52.5%).
* **Deep Insights:**
    * **Head-to-Head History:** Displays historical win counts between the two selected teams.
    * **Key Players:** Breakdowns of Batting and Bowling cores (e.g., Top Run Scorers and Wicket Takers) for strategic analysis.
* **Interactive UI:** Built with **Streamlit** for a seamless, user-friendly experience with a dark-mode stadium theme.
